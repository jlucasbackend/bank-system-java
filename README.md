# Sistema Bancário em Java

Projeto desenvolvido para praticar os fundamentos da linguagem Java através da simulação de operações bancárias em um ambiente de terminal.

## Funcionalidades
- Cadastro do nome do cliente
- Consulta de saldo
- Depósito de valores
- Saque de valores
- Menu interativo para navegação

## Tecnologias
- Java
- IntelliJ IDEA

## Conceitos praticados
- Variáveis e tipos de dados
- Estruturas condicionais (if/else)
- Estruturas de repetição (while)
- Estrutura de seleção (switch)
- Entrada de dados com Scanner
